## Open command prompt in the go workspace where server.go is located

    go mod init
    go mod vendor

## Login to heroku

    heroku login
    heroku create your-app-name

## Initialize git

    git init
    git add .
    git commit -m "heroku init"
    git push heroku master

## Check

    heroku open
